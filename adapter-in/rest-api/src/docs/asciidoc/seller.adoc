[[resources-seller]]
== Seller API

셀러 관리 API입니다.

[[resources-seller-register]]
=== 셀러 등록

`POST /api/v1/crawling/sellers`

새 셀러를 등록합니다.

==== Request Body

|===
| Field | Type | Required | Description

| `sellerName`
| String
| Yes
| 셀러명 (최대 100자)

| `mustItSellerName`
| String
| Yes
| 머스트잇 셀러명 (최대 100자)

| `sellerBaseUrl`
| String
| Yes
| 셀러 기본 URL
|===

==== Request Example

[source,http]
----
POST /api/v1/crawling/sellers HTTP/1.1
Host: localhost:8080
Authorization: Bearer {jwt-token}
Content-Type: application/json

{
  "sellerName": "테스트셀러",
  "mustItSellerName": "머스트잇셀러",
  "sellerBaseUrl": "https://example.com"
}
----

==== Response Example

[source,json]
----
{
  "success": true,
  "data": {
    "sellerId": 1,
    "message": "셀러가 등록되었습니다."
  },
  "error": null,
  "timestamp": "2025-01-15T12:00:00"
}
----

[[resources-seller-list]]
=== 셀러 목록 조회

`GET /api/v1/crawling/sellers`

셀러 목록을 페이징 및 필터링하여 조회합니다.

==== Query Parameters

|===
| Parameter | Type | Required | Description | Example

| `sellerName`
| String
| No
| 셀러명 필터 (부분 일치 검색, 최대 100자)
| `테스트`

| `mustItSellerName`
| String
| No
| 머스트잇 셀러명 필터 (부분 일치 검색, 최대 100자)
| `머스트잇`

| `statuses`
| List<String>
| No
| 상태 필터 목록 (다중 선택 가능: ACTIVE, INACTIVE)
| `statuses=ACTIVE&statuses=INACTIVE`

| `createdFrom`
| Instant
| No
| 생성일 시작 (ISO-8601)
| `2025-01-01T00:00:00Z`

| `createdTo`
| Instant
| No
| 생성일 종료 (ISO-8601)
| `2025-12-31T23:59:59Z`

| `page`
| Integer
| No
| 페이지 번호 (0부터 시작, 기본값: 0)
| `0`

| `size`
| Integer
| No
| 페이지 크기 (기본값: 20, 최대: 100)
| `20`
|===

==== Request Example

[source,http]
----
GET /api/v1/crawling/sellers?statuses=ACTIVE&page=0&size=20 HTTP/1.1
Host: localhost:8080
Authorization: Bearer {jwt-token}
----

==== Response Example

[source,json]
----
{
  "success": true,
  "data": {
    "content": [
      {
        "sellerId": 1,
        "sellerName": "테스트셀러",
        "mustItSellerName": "머스트잇셀러",
        "sellerBaseUrl": "https://example.com",
        "status": "ACTIVE",
        "createdAt": "2025-01-01T00:00:00Z"
      }
    ],
    "page": 0,
    "size": 20,
    "totalElements": 1,
    "totalPages": 1
  },
  "error": null,
  "timestamp": "2025-01-15T12:00:00"
}
----
