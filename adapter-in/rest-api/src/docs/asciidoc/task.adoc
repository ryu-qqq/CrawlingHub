== CrawlTask API

크롤 태스크 관리 API입니다.

[[resources-task-retry]]
=== 태스크 재시도

`POST /api/v1/crawling/tasks/{id}/retry`

실패한 크롤 태스크를 재시도합니다.

재시도 조건:

* FAILED 또는 TIMEOUT 상태의 태스크만 재시도 가능
* 최대 재시도 횟수(3회) 미만
